# myTab Chrome 扩展

个人网址导航新标签页 Chrome 扩展。

## 安装方法

1. 打开 Chrome 浏览器，访问 `chrome://extensions/`
2. 开启右上角的「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择 `extension` 文件夹
5. 打开新标签页即可使用

## 功能特性

- ✅ 分类管理网站书签
- ✅ 拖拽排序
- ✅ 右键菜单编辑/移动/删除
- ✅ 搜索网站
- ✅ 自定义背景/Logo/颜色
- ✅ 导入/导出配置
- ✅ 记住上次打开的分类
- ✅ 数据本地存储（chrome.storage）

## 从 Web 版本迁移

如果你之前使用 Web 版本，可以：
1. 在 Web 版本中导出配置
2. 在扩展中导入配置
